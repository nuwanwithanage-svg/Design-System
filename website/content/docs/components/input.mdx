## Overview

The Input component captures single-line text. It supports 2 sizes, 6 interactive states, and 3 affix configurations — 36 variants total.

## When to use

- Capturing short text (name, email, search query)
- Any single-line data entry in a form

## When not to use

- Multi-line text — use [Textarea](/docs/components/textarea) instead
- Selecting from a list — use a Select or Combobox

## Anatomy

1. **Container** — frame with background, border, corner radius
2. **Placeholder text** — hint shown when empty
3. **Value text** — user-entered content
4. **Leading icon** — optional icon (left side)
5. **Trailing icon** — optional icon (right side, e.g. clear button, unit)
6. **Focus ring** — shadow effect on focus

## Sizes

| Size | Height | PaddingH | Gap |
|------|--------|----------|-----|
| `sm` | 32px | 12px (spacing/4) | 8px (spacing/3) |
| `md` | 40px | 16px (spacing/5) | 8px (spacing/3) |

## States

| State | Background | Border | Text |
|-------|-----------|--------|------|
| `default` | `input/background/default` | `input/border/Default` | `input/text/placeholder` |
| `hover` | `input/background/default` | `input/border/hover` | `input/text/placeholder` |
| `focus` | `input/background/default` | `input/border/focus` | `input/text/value` |
| `error` | `input/background/default` | `input/border/error` | `input/text/value` |
| `disabled` | `input/background/disabled` | `input/border/disabled` | `input/text/disabled` |
| `readonly` | `input/background/disabled` | `input/border/Default` | `input/text/value` |

## Affix configurations

| Affix | Description |
|-------|-------------|
| `none` | Plain input, no icons |
| `leading-icon` | Icon on the left (e.g. search, user) |
| `trailing-icon` | Icon on the right (e.g. clear, eye toggle) |

## Accessibility

- Always pair an input with a visible `<label>`
- Use `aria-describedby` to link error messages
- Error state must not rely on color alone — include an error icon or message
- `readonly` inputs must communicate their state (not just visual muting)
