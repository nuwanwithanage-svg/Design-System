## Usage guidance

| Level | Use for |
|-------|---------|
| `sm` | Cards, image thumbnails, subtle floating elements |
| `md` | Dropdowns, select menus, date pickers, tooltips |
| `lg` | Modals, side sheets, dialogs |

<Callout type="info" title="Dark mode shadows">
Shadows appear at reduced opacity in dark mode. Increase shadow spread slightly or use border + shadow combos for better visibility.
</Callout>
