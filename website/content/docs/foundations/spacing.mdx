## Overview

The spacing system uses a named numeric scale from 0–11, covering 0px to 64px. Semantic aliases group these into meaningful categories.

## Usage guidance

Use spacing tokens for all padding, margin, and gap values. Do not use arbitrary pixel values.

```css
/* Correct */
padding: var(--spacing-4); /* 12px */
gap: var(--spacing-3);     /* 8px */
```

<Callout type="info" title="Component padding reference">
- **SM components** (h=32px): paddingH = spacing/4 (12px)
- **MD components** (h=40px): paddingH = spacing/5 (16px)
- **LG components** (h=48px): paddingH = spacing/6 (20px)
</Callout>
