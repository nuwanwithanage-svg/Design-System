## Overview

The colour system is built on two layers:

- **7 primitive palettes** — Red, Orange, Yellow, Dark Cyan, Green, Blue, Greyscale + Black/White
- **59 semantic tokens** — organized across 6 groups: Background, Text, Border, Icon, Focus, Input

<Callout type="info" title="Always use semantic tokens">
Use `background/brand/primary` in components, never `dark-cyan/80` directly. Semantic tokens communicate **intent** and allow global theme changes without touching components.
</Callout>

## Do / Don't

**Do** — use semantic tokens in components:

```css
background-color: var(--color-background-brand-primary);
```

**Don't** — use primitive tokens directly in components:

```css
/* Incorrect — do not use primitive tokens in components */
background-color: var(--color-dark-cyan-80);
```
